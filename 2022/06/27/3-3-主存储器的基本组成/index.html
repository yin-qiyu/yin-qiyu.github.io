3.3 主存储器的基本组成

主存储器

  • 主存储器分为:存储体、MAR（memory address register）、MDR（memory data register)

存储体由存储单元构成

存储单元由存储元构成

存储体存放二进制01

  • 字节和存储字的区别

字节：Byte

存储字：由硬件设计（如图存储字长为8）

  • 数据传输

MAR地址寄存器存储n位地址（意味着有2^n存储单元），分别给高低电平来控制不同的自选线，读取对应存储单元的信息

总容量 = 存储单元个数 存储字长 = 2^3 8bit = 8 1Byte =8B

  • 控制电路

只有MAR信号稳定后，译码器才会打开，控制电路保证电路稳定

  • 片选线

字母头上划线表示低电平有效

两根读写线：读写

一根读写：低写，高读

  • 存储芯片构造

驱动器：电信号放大部件

片选线：控制读取指定存储芯片的数据

  • 计算引脚

  • 存储芯片表示

存储单元✖️存储字长

  • 寻址

⭐️SRAM（双稳态触发器）和DRAM（栅极电容）

  • Dynamic Random Access Memory（动态）
  • Static Random Access Memory（静态）

区别

  • 核心区别：存储元不一样

dram：==栅极电容==存储信息

sram：==双稳态触发器==存储信息

  • 工作原理

dram（读写慢）：电容释放信息被破坏，==破坏性读出==，读出后应有==重写==操作（再生）

sram（读写快）：读出数据，触发器爆出稳定状态，==非破坏性读出，无需重写==

  • 制造成本

dram：集成度高，功耗低，成本低

sram：集成度低，功耗高，成本高

DRAM刷新

双译码器选通线数量为2^n/2^+2^n/2^根

如： 单译码器：2^8^=256，双译码器2^4^+2^4^=32

DRAM地址线复用技术

使用n/2条地址线分别将行地址和列地址送入缓冲器

行列地址分两次送，可以使地址线更少，芯片引脚更少

如今储存采用SDRAM：DDR3、DDR4

只读存储器ROM（Read Only Memory）

MROM–掩模式只读存储器 任何人不可重写

PROM–可编程只读存储器 写一次之后就不可以更改

EPROM–可擦除可编程只读存储器 可进行多次重写

UVEPROM–紫外线照射擦除所有信息

EEPROM–电擦除方式 擦除特定的字

Flash Memory–闪速存储器（U盘、sd卡）写的速度比读的速度更慢，每个存储元只需要单个mos管，位密度比ram高

SSD–固态硬盘 由控制单元+存储单元（flash）构成

手机辅存也是用flash芯片，但橡皮ssd使用的芯片集成度高，功耗低，价格贵
